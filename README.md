# Farmington User Search

Ths repository contains the github pages for the Farmington User Search application. This application allows users to search for their statistics and information on speedrun.com by using their API v2 information.

## Features

1. Search for users by username or ID by using the speedrun.com API v1.
2. Display user statistics and information retrieved from the API V2:
    - runs
    - games
    - categories
    - levels
    - platforms
    - regions
    - variable_values
    - variables
    - followed_games
    - challenges_accepted
    - challenges_runs
    - wrs
    - co_op_runs
    - runs_il
    - podiums
    - obsoletes
    - emulator_runs
    - splits_runs
    - 7th_places
    - wrs_il
    - runs_times
    - unique_verifiers
    - co_op_guests
    - games_with_wrs
    - co_op_partners
    - co_op_teams
    - misc_categories
    - categories_ils
    - categories_full_game
    - wrs_full_game
    - runs_full_game
    - personal_bests
    - non_misc_categories
    - co_op_non_guests
    - non_emulator_runs
    - non_splits_runs
